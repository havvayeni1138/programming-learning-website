 💻 Programming Learning Website
 📌 Project Overview

Programming Learning Website is a multi-page educational web project developed to teach basic programming languages and concepts through a clean and user-friendly interface.

This project was created to improve my front-end development skills and demonstrate my ability to design and structure a complete website using fundamental web technologies.

🎯 Project Purpose

The main purpose of this project is:

- To design a structured multi-page website
- To practice HTML page linking and navigation systems
- To build login and registration page interfaces
- To improve layout design and responsive structure skills
- To create a professional portfolio project for GitHub and CV

🚀 Features

- ✅ Home Page (Landing Page)
- ✅ Login Page
- ✅ Registration Page
- ✅ About Page
- ✅ Contact Page
- ✅ Navigation Menu Between Pages
- ✅ Multi-page Structure
- ✅ Image Integration
- ✅ Responsive Design Structure
- ✅ Clean and User-Friendly Interface
# 🛠️ Technologies Used

- HTML5
- CSS3
- Basic File Structure Management
- Git & GitHub (Version Control)
- GitHub Pages (Deployment)

 📂 Project Structure

- `index.html` → Main page  
- `giris.html` → Login page  
- `yenikayit.html` → Registration page  
- `hakkimda.html` → About page  
- `iletisim.html` → Contact page  
- `style.css` → Styling file  
- Image files → Visual content
